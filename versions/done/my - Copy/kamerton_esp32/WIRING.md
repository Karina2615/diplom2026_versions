## TTP229-BSF (16-кнопкова клавіатура)

| TTP229 pin | ESP32 pin | Примітка |
|---|---|---|
| VCC | 3.3V | живлення |
| GND | GND | земля |
| SCL | GPIO 16 | виход MCU (тактовий сигнал) |
| SDO | GPIO 17 | вхід MCU (дані від TTP229) |
| TP2 (OPT2) | GND | **ОБОВ'ЯЗКОВО** підтягнути до GND для 16-кнопкового режиму |

## BT-навушники — Перше підключення
1. Увімкни навушники в режим **парування** (зазвичай 5-секундне утримання кнопки живлення).
2. Відкрий `settings.h`, встанови `bt_device` = назва твоїх навушників  
   (або залиш порожнім для автопошуку).  
3. Завантаж прошивку. Пристрій автоматично шукає та підключається.  
4. Після першого підключення — автоматичне перепідключення при наступних запусках.

## Кнопки (TTP229) — Призначення по режимах

### Режим Generator (синтезатор)
| Кнопки | Дія |
|---|---|
| 1–7 | Ноти До–Сі |
| 8 | Октава ↑ |
| 9 | Октава ↓ |
| 10 | Гучність ↑ |
| 11 | Гучність ↓ |
| 12 | Наступний тембр |
| 13 | Вимкнути/увімкнути звук (Mute) |
| 16 | **Наступний режим** |

### Режим Arpeggio (арпеджіо)
| Кнопки | Дія |
|---|---|
| 1–7 | Акорди C-Dm-Em-F-G-Am-Bdim |
| 8 | Транспозиція ↑ |
| 9 | Транспозиція ↓ |
| 10/11 | Гучність ↑/↓ |
| 12 | Старт/Стоп |
| 16 | Наступний режим |
| Джойстик ↑/↓ | Вибір акорду |

### Режим Warmup (вокальна розспівка)
| Кнопки | Дія |
|---|---|
| 1–7 | Вибір послідовності 1–7 |
| 8/9 | Октава ↑/↓ |
| 10 | Грати |
| 11 | Стоп |
| 16 | Наступний режим |
| Джойстик ↑/↓ | Вибір послідовності зі списку |
| Джойстик кнопка | Грати виділену послідовність |

### Режим Recorder (запис/відтворення)
| Кнопки | Дія |
|---|---|
| 1 | Почати / Зупинити запис |
| 2 | Відтворити виділений запис |
| 3 | Зупинити відтворення |
| 4–7 | Швидкий вибір записів 1–4 |
| 16 | Наступний режим |
| Джойстик ↑/↓ | Прогортати список записів |
| Джойстик кнопка | Відтворити виділений запис |

### Режим Sampler (семплер, як PO-33)
| Кнопки | Дія |
|---|---|
| 1–15 | Семпл-пади (натисни = відтворення) |
| **Утримуй 16 + натисни 1–15** | Запис нового семплу на цей пад |
| **Відпусти 16** | Зупинити запис семплу |
| 16 (короткий тап) | Наступний режим |
| Джойстик ↑/↓ | Вибір пада (підсвічування) |
| Джойстик ←/→ | Налаштування висоти тону виділеного пада (±7 напівтонів) |
| Джойстик кнопка | Скинути висоту тону до нуля |

### Режим Tuner (тюнер)
Пасивний режим — показує детектовану частоту та відхилення в центах.  
Кнопка 16 = наступний режим.

## Файли на SD-карті
```
/REC_001.WAV  – записи (Recorder mode)
/REC_002.WAV
...
/SMP_001.WAV  – семпли (Sampler mode, пад 1–15)
...
/SMP_015.WAV
/SMP_PITCH.BIN – збережені налаштування висоти тону семплів
```

## Збереження налаштувань
Налаштування (режим, гучність, октава, тембр) автоматично зберігаються  
в **NVS (Non-Volatile Storage)** кожні 10 секунд та при кожній зміні режиму.  
Дані не стираються при вимкненні живлення.
